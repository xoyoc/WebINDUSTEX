## Colores principales

## Combinaciones sugeridas

### Modo claro
- **Fondo:** `#F5F5F5`
- **Texto principal:** `#2B2B2B`
- **Acentos:** `#E84C1F`
- **Detalles:** `#1F5E96`

### Modo oscuro
- **Fondo:** `#2B2B2B`
- **Texto principal:** `#F5F5F5`
- **Acentos:** `#F4B41A`
- **Detalles:** `#E84C1F`

---

## Aplicaciones recomendadas

| Aplicación | Colores a usar |
|-------------|----------------|
| Bordados textiles | `#E84C1F`, `#1F5E96`, `#F4B41A` |
| Uniformes | `#1F5E96` + `#2B2B2B` |
| EPP (Equipos de Protección Personal) | `#F4B41A` + `#E84C1F` (alta visibilidad) |
| Sitio web | Modo claro u oscuro según preferencia |
| Papelería | `#2B2B2B` + `#E84C1F` sobre `#F5F5F5` |

---

## Notas técnicas

- **Contraste accesible:** Todas las combinaciones cumplen con WCAG AA para textos normales (excepto amarillo sobre blanco).
- **Bordado:** Usar `#E84C1F` y `#1F5E96` con hilos polyester de alta resistencia.
- **EPP:** El amarillo `#F4B41A` cumple con estándares de alta visibilidad (ISO 20471).
